📱 테스트 URL 목록

📋 테스트 방법

  1. 데스크톱 테스트: 위 QR 코드를 스마트폰 카메라로 스캔하여 URL이 올바르게 열리는지 확인
  2. 모바일 브라우저 테스트: 스마트폰에서 직접 URL을 입력하여 접속
  3. 카메라 권한 테스트: QR 스캔 페이지에서 카메라 권한이 정상적으로 요청되는지 확인
  4. HTTPS 필요: 실제 스마트폰에서 카메라 기능을 사용하려면 HTTPS 연결이 필요합니다

🔧 HTTPS 로컬 서버 실행 방법:

HTTPS=true npm start (자체 서명 인증서 사용)

또는 ngrok을 사용한 터널링: ngrok http 3000
