## 🛠 **5단계: HTTPS 설정 (카메라 테스트용)**

### **로컬 HTTPS 개발 서버**
```bash
# 1. mkcert 설치 (Windows)
choco install mkcert

# 2. 로컬 CA 생성
mkcert -install

# 3. localhost 인증서 생성
mkcert localhost 127.0.0.1 ::1

# 4. React HTTPS 서버 실행
HTTPS=true SSL_CRT_FILE=localhost.pem SSL_KEY_FILE=localhost-key.pem npm start
```
